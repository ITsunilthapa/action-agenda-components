# Поиск Виджет

# Описание

Этот виджет отображает полнофункциональный механизм поиска обязательств.

Функции:

  1. свободный текстовый поиск

  2. Айти-целевой фильтр

  3. с фильтром

  4. фильтр по типу актера

  5. Тематический фильтр SCBD

  6. текстовый поиск для поиска доступных фильтров

  7. Envoke a filter change from external JS program by dispatching event $scbdFilterChange

  8. Поиск по URL-адресу для создания страниц или обмена

    а. сделать такую страницу страны, как: (opens new window) https://www.cbd.int/action-agenda/contributions/?filter=fr

    б. страница, посвященная цели: (opens new window) https://www.cbd.int/action-agenda/contributions/?filter=AICHI-TARGET-01

# Установить

# Виджет

Установить как копию и вставить виджет. Как видно здесь: (opens new window) https://www.cbd.int/action-agenda/contributions/

TIP

Параметры этого компонента могут быть переданы как атрибут в теге скрипта. См. Подробности о вариантах

# forceEnv

Сообщите компоненту, какой API использовать, передав: dev | stg | prod

dev : https://api.cbddev.xyz/api/actions

stg : https://api.stg.cbd.int/api/actions

prod : https://api.cbd.int/api/actions

# options

const validationMap =
{ forceEnv        : String,  /* default: production */
  basePath        : String,  /* default: '/' */
  locale          : String,  /* default: derives from the client automatically otherwise 'en' */
  api             : String,  /* default: api/cbd.int/api/actions */
  hostname        : String,  /* default: cbd.int - this is also used to derive the env if set. */
  listenExternally: Boolean, /* default: false - listen for js event $scbdFilterChange */
  preLoadFilter   : String,  /* preload a single filter by identifier 'AICHI-TARGET-01' */
  smTop           : String,  /* offset sticky header in small devices */
  mdTop           : String,  /* offset sticky header in medium devices */
  lgTop           : String   /* offset sticky header in large devices */
}
